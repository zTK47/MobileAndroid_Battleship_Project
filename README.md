# Battleship Android Project

## Team Members
- Davide Pham

## Project Description
This project is an Android client for the Battleship game.  
The app communicates with the provided server using a REST API and JSON.

## Current Status
Project initialized in Android Studio Panda.
Repository created and connected with Git.

## Planned Features
- Join a game with player name and game key
- Display two 10x10 boards
- Fire at the opponent
- Receive enemy moves
- Show hits, misses, and game over state

## Technologies
- Kotlin
- Android Studio Panda
- Git
- REST API
- JSON

Parts of this project may be developed with the help of AI tools. Generated code will be reviewed and adapted.
